## Key findings (durable lessons)

1. **Regime+holdout methodology is mechanism-specific.** It corrects a 2025-collapse + parameter-induced fragility pattern. Don't apply blindly to any "mediocre Sharpe" strategy. Use the 2025 OOS gate (champion 2025 < -10%) as a precondition.
2. **Two channels of regime adjustment can fight each other.** eod_technical's `direction_score` breadth filter is already a regime gate at the entry-day scale. Stacking a 100-250d NIFTYBEES SMA gate on top creates timescale conflict and whipsaw.
3. **Faster-cycling strategies (avg hold ≤60d) tolerate regime intervention worse** than slower ones. Force-exit on regime flip can fire before the position's natural exit thesis plays out.
4. **"Sharpe = CAGR / vol" is a different metric from rf-adjusted geometric Sharpe.** They differ by ~0.10-0.20 reliably. Always check which one a doc means; use engine canonical as the truth source.
